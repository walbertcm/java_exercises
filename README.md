# Java Exercises

Coleção de exercícios em Java

## 📚 Objetivo

Este repositório reúne exercícios práticos desenvolvidos para o ensino de programação em Java.  

## Observação

Cada arquivo contém um programa Java independente.

## ▶️ Como Executar

1. Copie o conteúdo de um dos arquivos `.txt`
2. Cole em um arquivo `.java` com o mesmo nome da classe
3. Compile e execute.
